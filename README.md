# Secure Password Generator (Python)

A simple CLI program that generates and validates secure passwords.

## Features

- Generate multiple random passwords
- Validate password security
- Enforces password rules
- Interactive menu

## Password Rules

Passwords must contain:

- 8–16 characters
- At least one uppercase letter
- At least one lowercase letter
- At least one number
- At least one symbol

## Example

MENU
1 - Generate password
2 - Validate password
3 - Exit
